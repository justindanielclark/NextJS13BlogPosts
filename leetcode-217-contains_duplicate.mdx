## The Problem

Neetcode Set: Arrays & Hashing
Neetcode Problem \#: 1\/9
Difficulty: Easy

Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.

#### Example 1:
- Input: nums = [1,2,3,1]
- Output: true
#### Example 2:
- Input: nums = [1,2,3,4]
- Output: false
#### Example 3:
- Input: nums = [1,1,1,3,3,4,3,2,4,2]
- Output: true
#### Constraints:
- 1 \<= nums.length \<= 10^5
- -10^9 \<= nums[i] \<= 10^9

## The Solution

First thing that comes to mind when I hear about checking for duplicates is a hash/set. Implementing a set and then checking if a number is in it should be easy enough, so lets do that:

```ts
function containsDuplicate(nums: number[]): boolean {
  const numSet = new Set<number>(nums);
  return numSet.size !== nums.length
};
```

Quick, easy, two liner.
